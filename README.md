**Healthcare Operations Dashboard**
**Project Overview**

This project analyzes healthcare operational data to monitor employee productivity, department performance, and accuracy using Microsoft Excel. The dashboard provides key performance indicators (KPIs) and visual reports to support operational decision-making.

**Business Objective**

The objective of this dashboard is to:

Monitor productivity across departments.
Track weekly operational performance.
Measure employee accuracy.
Identify top-performing employees.
Support data-driven decision-making.

**Tools Used**

Microsoft Excel
Pivot Tables
Pivot Charts
Conditional Formatting
Lookup Functions
KPI Dashboard

**Dashboard KPIs**

Total Cases Reviewed
Average Accuracy
Maximum Accuracy
Department Analysis
Weekly Analysis
Department vs Week Analysis
Employee Ranking

**Business Insights**

Compared department-wise productivity.
Monitored weekly operational trends.
Identified top-performing employees based on cases reviewed.
Evaluated accuracy levels across departments.
