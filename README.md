# Healthcare Operations Dashboard

## Project Overview

This project analyzes healthcare operational data to monitor productivity, departmental performance, and accuracy using Microsoft Excel

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Conditional Formatting
- KPI Cards

## Dashboard Features

- Total Cases Reviewed
- Average Accuracy
- Maximum Accuracy
- Department-wise Analysis
- Weekly Productivity Analysis
- Department vs Week Comparison
- Employee Performance Ranking

## Business Insights

- Compared productivity across departments.
- Monitored weekly operational performance.
- Identified top-performing employees based on cases reviewed.
- Evaluated accuracy trends to support quality monitoring.

- Data Cleaning
- Data Analysis
- Dashboard Design
